# Research Sources

The project uses primary manufacturer documentation wherever it is available.
Copied datasheets, teardown camera originals, temporary web captures, and
vendor CAD archives are kept in the local project archive rather than mirrored
in GitHub. This keeps the repository small and avoids republishing files with
unclear redistribution terms.
